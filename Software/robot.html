Project Spyn Robot (2023)

Description

For this project we were tasked to construct and program a robot in groups of four. The robot had to be able to solve a simple one line maze, recognize colors, and enter a state of manual control to pick up a "passenger". The project taught me numerous things, such as the difficulties of programming in physical spaces, how to work in a team, and how to quickly learn new systems, as I had to learn how to use Matlab and the Lego Mindstorm robot kit.

What I Learned:

As mentioned above, this project was made using MatLab. In fact, it was my first time ever using the program, meaning I learned a lot from this project. Additionally, it was my first time programming any hardware, and it taught me how difficult doing so can be. It is also my only foray into mechanical engineering, even if it was just Lego.
